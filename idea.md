\# Project Idea



\## Inspiration



The project was inspired by classic turn-based strategy games such as Heroes of Might and Magic.



The goal was not to recreate these games, but to design a small strategy game that demonstrates programming concepts such as game logic, artificial intelligence, resource management, and user interface development.



\## Goal



The objective was to create a playable turn-based strategy game where:



\- the player manages resources,

\- recruits different unit types,

\- builds mines,

\- attacks the enemy,

\- defeats the enemy castle.



\## Main Features



\- Turn-based gameplay

\- Gold economy

\- Three unit types

\- Enemy AI

\- Combat system

\- Victory and defeat conditions

\- Combat log

\- Restart after game over



\## Challenges



During development several challenges appeared:



\- balancing combat mechanics,

\- designing simple enemy AI,

\- creating a clean user interface,

\- organizing the project with Git and GitHub.



These problems were solved incrementally by testing, debugging and improving the implementation.
