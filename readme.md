\# Kingdom ClashKingdom Clash is a simple turn-based strategy game developed in \*\*Godot 4\*\* using \*\*GDScript\*\*.The project was created as a university programming assignment to demonstrate object-oriented thinking, game logic, UI programming, AI behavior, and version control with Git.\---\## Features\- 💰 Gold economy\- ⛏️ Build mines to increase income\- ⚔️ Recruit Soldiers\- 🏹 Recruit Archers\- 🪨 Recruit Catapults\- 🎯 Select attack target\- 🤖 Enemy AI\- 📜 Combat log\- 🏆 Victory / Defeat conditions\- 🔄 Restart game with \*\*R\*\*\---\## GameplayEach turn the player receives gold from mines, recruits units, attacks the enemy and ends the turn.The enemy AI also collects gold, recruits units and attacks automatically.The objective is to destroy the enemy castle before your own castle is destroyed.\---\## Technologies\- Godot Engine 4.7\- GDScript\- Git\- GitHub\---\## Project Structure```main.gdmain.tscnREADME.mdidea.mdresources.md```\---\## How to Run1\. Open the project in Godot 4.7.2\. Run `main.tscn`.3\. Recruit units.4\. Build mines.5\. Attack the enemy.6\. Destroy the enemy castle.\---\## Future ImprovementsPossible future extensions include:\- Better AI\- More unit types\- Castle upgrades\- Random events\- Sound effects\- Better graphics
